# Docs Meta Scaffold

This app is the parallel Vite 8 framework-based docs site scaffold.

Current scope:
- custom Vite 8 app with a client and server environment
- Arrow SSR and hydration wiring
- minimal placeholder routes only

Intentionally not done here:
- final UI
- content migration from the legacy `docs/` app
- design system or styling work

Commands:
- `npx pnpm@10.14.0 dev:meta`
- `npx pnpm@10.14.0 build:meta`
- `npx pnpm@10.14.0 preview:meta`
